# 재귀호출 [recursive_call]


### 유클리드 호제법

```
일반적인 최대공약수를 구하는 방법은 두 수를 소인수 분해한 후 공통된 소수를 찾아서 곱해주면 된다.
```

```
두 정수의 최대 공약수(greatest common divisor)를 재귀적으로 구하는 방법   
두 정수를 직사각형의 두 변의 길이라고 생각하면 두 정수의 최대공약수를 구한다.
- 짧은 변의 길이를 한 변으로 하는 정사각형으로 직사각형을 채운다.
- 남은 직사각형에 대해 위와 같은 작업을 반복한다.
- 마지막으로 정사각형으로 구성되었을 때의 변의 길이가 최대공약수이다.

두 정수 x,y의 최대공약수를 gcd(x, y)로 표기한다.
x = az 와 y = bz를 만족하는 정수 a,b 와 최대의 정수 z가 존재할 때 z를 gcd(x,y)라고 할 수 있다.
다시말해 최대공약수는 y가 0이면 x이고, y가 0이 아니면 gcd(y, x % y)로 구한다.
```

```
[x, y]의 최대공약수

1. 큰수를 작은수로 나눈다. (x > y)
2. 나누는 수를, 나머지로 계속 나눈다.
  나머지가 0이 될 때까지 반복한다. 나머지가0이 되면 나누는 수가 최대공약수이다.
```

```java
static int gcd(int x, int y) {
        if (y == 0) {
            return x;
        } else {
            return gcd(y, x % y);
        }
    }
```
