# straight_insertion_sort [단순 삽입 정렬]

```
선택한 요소를 그보다 더 앞쪽의 알맞은 위치에 '삽입하는' 작업을 반복하여 정렬한다.
단순 선택 정렬은 값이 가장 작은 요소를 선택해 알맞은 위치로 옮긴다는 점이 다르다.
```

```
매번 정렬된 배열과 비교할 배열을 비교하여 삽입한다.
매번 swap을 하면서 비교할 대상인 값이 정렬된 배열내에서 자신보다 작은 값을 찾을 때까지 swap한다.
```

```
정렬된 열의 왼쪽 끝에 도달한다.
tep보다 작거나 같은 key를 갖는 항목a [j]를 발견한다.

j가 0보다 크다.
a[j - 1]값이 tmp보다 크다.
```

##### 교환횟수는 n^2/2 회 이다.
##### 시간복잡도는 O(n^2)이다.


```java
class InsertionSort {
    public static void main(String[] args) {

    }


    static void insertionSort(int[]a, int n) {
        /**
         * 정렬의 시작은 왼쪽부터 시작한다.
         * for문은 1부터 시작해 요솟수n보다 작을 때 까지 순회한다.
         * 1부터 시작하는 이유는 정렬배열 기준 왼쪽으로 비교하기 때문이다.
         * [0][1][2][3][4][5]...[n - 1]
         * 0 과 1을 비교하기 위해서
         * tmp값을 a[i] -> a[0]으로 초기화한다. 비교할 값
         * 그 다음 for문은 j의 값을 i로 초기화하고, j는 0보다 크고
         * a[j - 1] > tmp 보다 크다면 :: 비교할 값에서 이전값이 tmp보다 크다면
         * 오름차순 기준 swap해야한다.
         *
         */
        for (int i = 1; i < n; i++) {
            int j;
            int tmp = a[i];
            for (j = i; j > 0 && a[j - 1] > tmp; j--) {
                a[j] = a[j - 1];
                a[j] = tmp;
            }
        }
    }
}
```
