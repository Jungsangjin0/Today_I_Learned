# 하노이의 탑

```
작은 원반이 위에, 큰 원반이 아래에 위치할 수 있도록 원반을 3개의 기둥 사이에서 옮기는 문제
처음에 원반이 놓인 기둥을 "시작 기둥", 목적지의 기둥을 "목표 기둥" 남은 중간의 기둥을 "중간 기둥" 이라 칭한다.
원반 1과 원반2가 겹친 것을 "그룹"이라 부른다.
가장 큰 원반을 최소의 단계로 목표 기둥으로 옮기려면 가장 먼저 "그룹"을 중간 기둥으로 옮겨야 한다.
원반이 4개일 때도 동일하다. 원반1,2,3을 그룹으로 중간기둥으로 옮기고 가장 큰 원반을 목표기둥으로 이후 그룹을 그 위에 쌓으면 된다.
```
```java
static void move(int no, int x, int y) {
        if (no > 1) {
            /**
             * 6인 이유
             * 기둥 번호를 정수 1,2,3 으로 나타낸다. 기둥 번호의 합이 6이므로
             * 시작 기둥, 목표 기둥이 어느 기둥이더라도 중간 기둥은 6 -x -y로 구할 수 있다.
             * 
             */
            move(no - 1, x, 6 - x - y);

            System.out.println("원반 [" + no + "]를" + x + "기둥에서" + y + "기둥으로 옮김");

            if (no > 1) {
                move(no - 1, 6 - x - y, y);
            }
        }
    }
```
