# modifiers

```
v-model에 modifiers 로 lazy, number, trim을 사용할 수 있다
[ v-model.lazy ]
[ v-model.number ]
[ v-model.trim ]
```

```
trim은 공백을 없애준다. [ 시작과 끝 ]
number은 입력값을 Number로 typeCasting 해준다. [ parseFloat 은 아님 ]
lazy는 v-model의 기본 값으로 모든 input event가 끝난 후에 값이 변한다.
```
